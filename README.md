##Goals

- CMS with Mustache
- API for Frontend & Mobile
- Responsive UI

#Setup

1.Install

    pip install -r requirements.txt

2.Setup Database

    python manage.py syncdb
    python manage.py migrate

3.Run

    python manage.py runserver
